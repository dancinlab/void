Void VT Key Encoder - WebAssembly Example

This example demonstrates encoding key events into terminal escape sequences using the Void VT WebAssembly module.

⚠️ Warning: This is an example of the libvoid-vt WebAssembly API. The JavaScript keyboard event mapping to the libvoid-vt API may not be perfect and may result in encoding inaccuracies for certain keys or layouts. Do not use this as a key encoding reference.
